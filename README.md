## 🐾 프로젝트 소개
**꼬순내 신당**은 반려동물의 생년월일만 입력하면  
성격, 습관, 전생, 궁합, 오늘의 기분과 간식까지 귀엽고 신비롭게 알려주는 운세 서비스입니다.  

LLM 기반 자연어 생성을 통해, 보호자와 반려동물 사이의 감정 교감을 더 깊고 특별하게 만들어줍니다.  
단순한 운세 풀이를 넘어, 반려동물을 더 잘 이해하고 사랑할 수 있도록 도와주는  
**국내 최초의 반려동물 전용 신점 기반 콘텐츠 플랫폼**입니다.

- 귀엽고 감성적인 UI와 세계관  
- 현실적인 행동 분석 + 서사형 전생 스토리  
- 오늘의 간식 추천, 장난감 MBTI, 케미 해석 등  

작지만 사랑스러운 디테일로 하루를 채워드립니다.

## ✨ 핵심 기능
- **반려동물 사주 분석**  
  사주 기반의 해석으로 반려동물의 성격, 습관, 사랑 표현 방식,  
  사주 기반 MBTI 스타일 및 장난감 추천까지 포함된 맞춤형 운세 리포트 제공

- **반려동물 궁합 확인**  
  두 반려동물 간의 사주 궁합을 분석하여 감정 흐름, 거리감, 놀이 상호작용, 루틴 조화 등을  
  구체적이고 현실감 있게 해석하고, 케미 행동 묘사와 속마음 메시지, 공동 장난감 추천까지 포함한  
  귀엽고 생생한 궁합 콘텐츠 제공

- **반려인 궁합 분석**  
  보호자와 반려동물의 사주를 바탕으로 상호작용 스타일, 놀이 궁합, 루틴 케미를 해석하고  
  오늘의 기분 흐름과 함께 일상 속 행동 팁을 센스 있게 제안

- **반려동물 전생 탐색**  
  반려동물의 전생, 애칭, 살던 곳, 마지막 기억, 남겨진 흔적까지  
  사주 기반으로 구성된 감성적이고 신비로운 이야기 제공  
  귀엽고도 촉감 있는 표현을 통해 보호자에게 감정적 몰입을 선사

- **오늘의 기분 & 간식 추천**  
  사주의 일간 흐름을 기반으로 반려동물의 감정 상태와 어울리는 간식을 추천하여  
  하루를 더 사랑스럽게 만들어주는 콘텐츠 제공

## 🏗️ 아키텍처 구조
### 🧩 마이크로서비스 아키텍처  
각 기능(사주, 궁합, 전생)은 독립적인 마이크로서비스로 구성되어 있어 개별 배포와 유지보수가 유연합니다.

### ⚖️ 로드 밸런싱  
Netflix Eureka를 통한 서비스 디스커버리 및 로드 밸런싱 구현:
- 트래픽 자동 분산으로 안정적인 처리
- 인스턴스 등록/해제 자동화
- 장애 복원력 향상

### 🤖 AI 기반 운세 콘텐츠 생성  
OpenAI GPT 기반 LLM API를 활용하여 콘텐츠를 생성:
- 반려동물 특성 기반 맞춤 운세 제공
- 자연스럽고 몰입감 있는 텍스트 출력
